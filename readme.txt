Notes
~~~~~

Following files are included:
    mglobals.js     - Some global functions used throughout
    mdecimal.js     - To provide the MDecimal class object
    mdatetime.js    - To provide the MDate, MDateTime and Mtime class objects

~~~~~~~~~~~~
